## Project Description
This project predicts electricity demand using weather data from 2021–2024.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Random Forest Regression
- Data Visualization

## Dataset Features
- Temperature
- Humidity
- Wind Speed
- Atmospheric Pressure

## Model Used
Random Forest Regression

## Results
The Random Forest model performed better than Multiple Linear Regression with lower prediction error.
